# 🚦 AI Traffic Optimization System

## 📌 Overview
This project uses computer vision and AI concepts to detect vehicles and dynamically simulate traffic signal control based on traffic density.

## 🎯 Objective
To reduce traffic congestion by analyzing real-time vehicle count and adjusting traffic signals intelligently.

## 🛠️ Technologies Used
- Python
- OpenCV
- Computer Vision

## ⚙️ Features
- Vehicle detection using Haar Cascade
- Real-time traffic density analysis
- Dynamic signal simulation
- Live video processing

## 🧩 How It Works
- Video input is processed frame by frame
- Vehicles are detected using a pre-trained classifier
- Number of vehicles is counted
- Signal is adjusted based on traffic density

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py
